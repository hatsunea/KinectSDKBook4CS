ドキュメント構造

ドキュメントのツリー構造

ドキュメント関連のクラスは Hisui.Core ネームスペースに定義されています。

ドキュメント関連のクラス図

上記クラスが構成するツリー構造

各クラス/オブジェクトの簡単な説明

Document ドキュメント。ツリー構造のルート。Hisui.Core.Document.Currentで現在アクティブなのドキュメントを取得できる。
IEntry エンティティを一つ格納するエントリ。CADEL系CADのHDR（ヘッダ）に相当する。
IEntryCollection IEntry を複数保持するコレクション。IEntry を ID で管理する。
entity 任意のオブジェクトをエンティティとしてIEntryに登録することが出来る。 典型的なエンティティとしては、曲線や曲面、メッシュといった幾何要素が挙げられる。
element 任意のオブジェクトをエレメントとしてIEntryに登録することが出来る。 主となるのは entity であり、element はそれを補足・補助する付加的な要素である。 典型的なエレメントとしては、エンティティの表示要素（Hisui.Graphics.IScene）が挙げられる。
